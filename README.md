# GeCL Launcher

GeCL Launcher is a personal Minecraft launcher developed for learning Microsoft authentication and Minecraft launcher development.

## Features

- Microsoft Account Login
- Minecraft Java Edition Authentication
- Fabric Modpack Launching
- Local Game Management

## Purpose

This project is intended for educational and personal use only.

The launcher uses the official Microsoft OAuth authentication flow and the official Minecraft authentication services. It does not bypass authentication, licensing, or security mechanisms.

Under active development.
